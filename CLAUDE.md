# Claude Code 프로젝트 지시사항

## 모호한 요청 처리

모호하거나 불명확한 요청을 받았을 때:

1. **즉시 작업하지 말고** 먼저 명확화 질문을 한다
2. 다음 항목들을 확인한다:
   - 구체적인 목표가 무엇인지
   - 어떤 파일/범위를 대상으로 하는지
   - 원하는 결과물의 형태
   - 제약 조건이나 선호하는 방식
3. AskUserQuestion 도구를 적극 활용한다
4. 추측하지 말고 물어본다

## 질문 예시

- "어떤 파일을 수정할까요?"
- "기존 코드를 유지할까요, 대체할까요?"
- "테스트도 작성할까요?"
- "에러 처리는 어떤 방식을 원하시나요?"
